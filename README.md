# Hologram OS

A gesture-controlled holographic operating system built with Python.

## Features

- Hand gesture navigation
- Spatial file explorer
- Pinch grab interaction
- Two-hand expand to open folders
- Swipe navigation
- Drag and drop files
- Depth-based UI

## Technologies

- Python
- OpenCV
- MediaPipe
- Pygame
- Spatial UI Engine

## Controls

Pinch → Grab object  
Two-hand expand → Open folder  
Swipe left → Navigate back  
Fist → Drag file  
Swipe up/down → Scroll  

## Vision

This project explores the future of spatial computing interfaces similar to:

- Iron Man holographic UI
- Apple Vision Pro interfaces
- Minority Report gesture systems
